# mycpu

一个基于 **RV32I** 的简易 CPU 模拟器项目，支持运行裸机程序，并附带几个终端交互示例，其中包括多关卡迷宫游戏。

## 功能简介

这个项目目前包含：

- RV32I 指令集模拟执行
- UART 终端输出
- 键盘输入
- 定时器支持
- 裸机程序加载与运行
- 终端小游戏示例

目前仓库里最有代表性的演示程序是：

- 迷宫游戏（Maze Game）
- 其他基础固件示例（位于 `fw/`）

## 项目结构

```text
mycpu/
├── fw/               # 裸机固件 / 游戏程序
├── src/              # RV32I 模拟器源码
├── build/            # 构建输出目录（生成后出现）
├── CMakeLists.txt    # 模拟器构建配置
├── run_maze.sh       # 一键编译并运行迷宫游戏
